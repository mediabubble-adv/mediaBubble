# Platform Specifications for Masri Content

## Character Counts

| Platform | Optimal | Max | Notes |
|----------|---------|-----|-------|
| Facebook post | 80–250 | 5,000 | Arabic is 35% longer, adjust down |
| Instagram caption | 50–150 | 2,200 | Keep scrolling; long text hidden behind "more" |
| TikTok caption | 20–50 | 300 | Short, punchy; voiceover does the work |
| LinkedIn post | 100–300 | 3,000 | Professional; use L3–L4 |
| Tweet/X | 40–100 | 280 (Arabic chars) | Arabic tweets use fewer chars for same meaning |
| WhatsApp broadcast | 30–100 | 1,024 | Personal, direct |

## Image-to-Text Ratio

| Platform | Image/Video | Text | Ideal Ratio |
|----------|-------------|------|-------------|
| Instagram | Visual hero | 30–40 chars overlay | 80% visual, 20% text |
| Facebook | Image + text | 50–80 chars overlay | 60% visual, 40% text |
| LinkedIn | Text first | 80–150 chars overlay | 50% visual, 50% text |
| TikTok | Video | On-screen captions only | 90% video, 10% text |

## Hashtag Strategy by Platform

| Platform | Count | Position | Language |
|----------|-------|----------|----------|
| Instagram | 5–10 | End of caption or first comment | Arabic + English |
| Facebook | 2–3 | End of post | Arabic only |
| LinkedIn | 3–5 | End of post | Arabic + English |
| TikTok | 3–5 | In caption | Trending Arabic + English |
