# Agent Orchestration — Quality Gate Checklist

## G1 — Grammar Gate

- [ ] No verb conjugation errors (person/number/gender agreement)
- [ ] No case ending errors in MSA content
- [ ] Idafa (construct) phrases properly formed
- [ ] Gender agreement across adjectives and nouns
- [ ] Relative clauses properly introduced (الذي / التي / الذين)
- [ ] Hamza spelling consistent (wasl vs qat')

## G2 — Register Gate

- [ ] Dialect consistent throughout (no mixed L2 → L4 mid-text)
- [ ] L1–L5 level matches platform target
- [ ] Code-switching justified (e.g., tech terms, religious quotes)
- [ ] English loanwords naturalized or quoted
- [ ] Politeness level matches audience relationship

## G3 — Cultural Gate

- [ ] No red-line topics triggered inadvertently
- [ ] Imagery culturally appropriate for ALL target regions
- [ ] Religious references accurate and in proper register
- [ ] National days and observances correctly referenced
- [ ] Gender sensitivity: pronouns, imagery, address norms
- [ ] Sectarian neutrality unless specifically targeting

## G4 — Typography Gate

- [ ] Arabic fonts selected render correctly on target platform
- [ ] Diacritics visible at target font size
- [ ] RTL layout tested in browser/dev tools
- [ ] Bidirectional content verified (e.g., phone numbers, prices)
- [ ] Line height sufficient for diacritic clearance

## G5 — SEO Gate

- [ ] Target keywords present in title and first paragraph
- [ ] Meta description written and under 160 chars
- [ ] Open Graph tags set (og:title, og:description, og:image)
- [ ] h1–h3 structure clear for Arabic reader
- [ ] alt text on images in Arabic
- [ ] URL slug Arabic or Latin (consistent)
- [ ] Canonical URL set

## G0 — Brief Gate

- [ ] Target audience defined (country, age, platform)
- [ ] Content objective stated (inform / persuade / entertain / convert)
- [ ] KPIs defined (engagement / conversion / reach / retention)
- [ ] Brand voice constraints listed
- [ ] Competitor references (if any)
- [ ] Timeline and dependencies mapped
