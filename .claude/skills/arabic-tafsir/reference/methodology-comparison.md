# Tafsir — Methodology Comparison: Ma'thur vs Ra'y vs Contemporary

## Primary Methodologies at a Glance

| Aspect | Ma'thur (مأثور) | Ra'y (رأي) | Contemporary |
|--------|----------------|------------|-------------|
| Source | Quran → Hadith → Sahaba → Tabi'un | Ratio + Linguistics + Principles | All of the above + modern context |
| Authoritative | Highest (chain-based) | Varies (depends on scholar's ijtihad) | Moderate (depends on approach) |
| Risk | Overlooking meaning context | Personal bias (هوى) | Outdated social assumptions |
| Best for | Aqidah, fiqh rulings | Linguistic depth, philosophical questions | Modern relevance, da'wah |
| Length | Long (Tabari) to medium (Ibn Kathir) | Very long (Razi) to concise (Baydawi) | Medium (Qutb) to long (Ibn 'Ashur) |

## The Spectrum

```
Ma'thur (strict) ←————————→ Ra'y (pure)
     Tabari     Ibn Kathir     Baydawi     Zamakhshari     Razi

Contemporary approaches sit across the whole spectrum:

  Qutb (Zilal)           → Literary + Movement (leans Ra'y)
  Zuhayli (Munir)        → Legal + Comparative (leans Ma'thur)
  Ibn 'Ashur (Tahrir)    → Maqasid + Linguistic (balanced)
```

## How to Identify Methodology from the Text

| Marker | Indicates |
|--------|-----------|
| "حدثنا... عن..." (chain of narration) | Ma'thur — Tabari, Ibn Kathir |
| "قال أهل المعاني..." | Linguistic — Zamakhshari, Baydawi |
| "فإن قيل... قلنا..." (objection/response) | Ra'y — Razi, Abu al-Su'ud |
| "وقال بعض المفسرين..." | Eclectic — Alusi, Ibn 'Ashur |
| "في هذا العصر..." | Contemporary — Qutb, Zuhayli |

## Applied Example: Surah al-Ikhlas (112)

| Tafsir | Interpretation |
|--------|---------------|
| Tabari | Allah is one — no partners; refutes trinitarians and polytheists |
| Zamakhshari | Linguistic analysis of "أحد" vs "واحد"; why "أحد" is unique |
| Razi | 23 philosophical arguments against multiplicity in divine essence |
| Qutb | Emotional, literary, calls to tawhid-centered life |
| Ibn 'Ashur | Maqasid: the surah establishes theological foundation for Islamic society |
