# Arabic Calligraphy in Brand Applications

## Overview

Calligraphy is not just a logo — it's a brand system element that can appear across every touchpoint. This file covers how Arabic calligraphy applies across different media: packaging, signage, print, digital, environmental, and merchandise.

## Touchpoint Matrix

| Touchpoint | Calligraphy Role | Key Considerations |
|------------|-----------------|-------------------|
| Logo / Wordmark | Primary brand identifier | Scalability, legibility at all sizes |
| Packaging | Hero element or accent | Production constraints, material interaction |
| Signage | Wayfinding or brand statement | Viewing distance, lighting, material |
| Print (brochures, ads) | Headline or decorative | Ink traps, paper texture, color registration |
| Digital (web, app) | Header or accent | Screen rendering, loading performance |
| Social media | Profile image, cover, templates | Small sizes, animation, platform specs |
| Video / Motion | Logo reveal, titles, transitions | Easing, timing, legibility mid-motion |
| Merchandise | Embellishment, decoration | Embroidery minimums, print method limitations |
| Environmental | Wall murals, architectural | Scale, installation method, material |
| Stationery | Letterhead, business cards | Print method, foil stamping limits |

## 1. Packaging Applications

### Where Calligraphy Appears on Packaging

| Position | Impact | Considerations |
|----------|--------|----------------|
| Front face (hero) | Highest visibility | Must work with product shape and window |
| Back face | Secondary | Integration with legal text, ingredients |
| Side panel | Brand continuity | Repeat pattern, abbreviated mark |
| Cap / Closure | Close-up detail | Small surface, 3D curve |
| Inner packaging | Surprise/unboxing | Material limitations, food safety |
| Seal / Sticker | Trust marker | Small size, peel application |

### Material & Production Considerations

| Material | Calligraphy Method | Notes |
|----------|-------------------|-------|
| Cardboard / Paper | Print (offset, digital), foil stamp, emboss | Crisp lines, fine details possible |
| Glass | Screen print, etch, label | Consider curvature distortion, opacity |
| Plastic / Polymer | Print, emboss, in-mold decoration | Shrinkage, stretching during application |
| Metal | Etch, deboss, print | Hard surface, limited color |
| Fabric | Embroidery, screen print, heat transfer | Thread count minimums, stretch |
| Wood | Laser engrave, print, paint | Grain direction, burning vs cutting |

### Foil Stamping Guidelines

| Aspect | Recommendation |
|--------|----------------|
| Minimum stroke width | 0.5pt (0.18mm) |
| Minimum gap between strokes | 1pt (0.35mm) |
| Maximum foil area | 70% of surface (risk of peeling) |
| Best scripts for foil | Kufi (strong lines), simplified Naskh |
| Avoid for foil | Diwani thin hairlines, fine decorative details |
| Registration tolerance | ±0.5mm |

### Embossing / Debossing

| Aspect | Recommendation |
|--------|----------------|
| Minimum stroke width | 1pt (0.35mm) |
| Recommended depth | 0.3-0.8mm (deeper for larger surfaces) |
| Best scripts | Bold Kufi, Thuluth (strong contrast) |
| Avoid | Fine scripts, close letter spacing |
| Paper weight minimum | 300gsm for good definition |

## 2. Signage & Environmental

### Outdoor Signage

| Factor | Consideration |
|--------|---------------|
| Viewing distance | Letter height = distance / 30 (minimum) |
| Lighting | Back-lit, front-lit, or halo-lit affects legibility |
| Material | Acrylic, aluminum composite, PVC, neon/LED |
| Script for distance | Bold Kufi, simplified Thuluth |
| Script to avoid | Diwani with fine details, complex flourishes |

### Indoor Signage

| Type | Calligraphy Use | Recommendation |
|------|----------------|---------------|
| Lobby / Reception | Hero brand mark in large format | Thuluth or contemporary statement piece |
| Directional | Wayfinding | Clear Kufi or Naskh for readability |
| Department | Section identifiers | Custom wordmarks per department |
| Quote / Art | Inspirational or brand values | Decorative, artistic calligraphy |
| Exit / Safety | Standard signs (non-calligraphy) | Use legible typeface, not calligraphy |

### Monument Signs

Calligraphy on freestanding monument signs needs:
- Weather resistance (UV, heat, dust)
- Structural integrity of letters (no thin unsupported strokes)
- Night visibility (internal or external lighting)
- Reading from vehicles (larger is better)

### Wall Murals

**Interior:**
- Paint, vinyl, or CNC-cut materials
- Can be highly detailed
- Lighting can be directional/dramatic
- Interactive potential

**Exterior:**
- Weather-resistant paints, materials
- Larger scale, simplified details
- Structural surface considerations
- Community context important

## 3. Print Applications

### Business Cards

| Element | Recommendation |
|---------|----------------|
| Logo placement | Front, centered or top-aligned |
| Name | Calligraphy name is powerful but must be readable — use simplified script |
| Details | Legible typeface, not calligraphy |
| Foil | Gold or copper foil for luxury |
| Emboss | Logo-only, not details |

### Brochures & Collateral

| Use | Calligraphy Approach |
|-----|---------------------|
| Cover headline | Display calligraphy as hero element |
| Section dividers | Decorative calligraphy page markers |
| Pull quotes | Decorative initial letters (drop caps) |
| Body text | Typeface, not calligraphy |
| Back cover | Smaller, elegant wordmark |

### Print Production Considerations

| Factor | Calligraphy Implication |
|--------|------------------------|
| Paper texture | Rough paper can distort fine hairlines |
| Ink spread (dot gain) | Thin strokes can fill in on absorbent paper |
| Pantone (spot color) | One-color calligraphy is distinctive, production-friendly |
| CMYK registration | Multiple plates can misalign on fine details |
| UV coating | Can add depth to calligraphy, but may cause glare |
| Varnish (spot) | Apply to calligraphy only for tactile contrast |

## 4. Digital Applications

### Website

| Element | Calligraphy Use | Technical Considerations |
|---------|-----------------|------------------------|
| Hero logo | Custom wordmark | SVG format, responsive sizing |
| Page headings | Calligraphic display type | Font subsetting, variable font if available |
| Section dividers | Decorative calligraphy | SVG or CSS — minimize HTTP requests |
| Testimonials / pull quotes | Calligraphic initial | One-off character as CSS background |
| Background | Subtle watermark pattern | Low opacity, large SVG or CSS pattern |

**Performance:**
- Calligraphy images should be SVG (small, sharp, responsive)
- Avoid PNG for large decorative elements
- Lazy-load hero calligraphy? No — it's above the fold
- Preload the calligraphy image if it's a single SVG
- Keep file size under 50KB for main logo

### Mobile App

| Element | Calligraphy Use | Limitation |
|---------|-----------------|------------|
| App icon | Simplified wordmark or letterform | 32px rendering — must be bold |
| Splash screen | Full wordmark with animation | Brief display, load fast |
| Onboarding | Calligraphy accents | Keep small, avoid over-design |
| Navigation | Avoid calligraphy for nav items | Use legible system typeface |
| Notifications | Never use calligraphy | Must be instantly readable |

### Social Media

| Platform | Calligraphy Use | Specs |
|----------|-----------------|-------|
| Profile image | Simplified letter mark | 180×180px minimum |
| Cover / header | Full wordmark | Platform-specific dimensions |
| Post graphics | Calligraphy headline | 1080×1080px (IG), 1200×630px (FB/LinkedIn) |
| Stories | Animated calligraphy reveal | 9:16 vertical, 5-15 seconds |
| Video thumbnails | Calligraphy on dramatic background | Bright contrast, readable at small thumbnail |

## 5. Merchandise & Promotional

### Embroidery

| Factor | Recommendation |
|--------|----------------|
| Minimum stroke width | 2mm (thicker is safer) |
| Minimum letter height | 10mm |
| Best scripts | Bold Kufi, simplified Naskh |
| Avoid | Thin hairlines, dense Diwani, tiny diacritics |
| Thread types | Rayon (sheen), polyester (durable), metallic (luxury) |
| Stitch types | Satin stitch (thick lines), fill stitch (large areas) |
| Placement | Left chest, center, sleeve — flat surfaces best |

### Screen Printing

| Factor | Recommendation |
|--------|----------------|
| Minimum stroke | 0.5pt for fine detail, 1pt recommended |
| Mesh count | Lower (110-160) for thick inks, higher (230-305) for detail |
| Best scripts | Kufi, simplified Naskh |
| Colors per design | 1-3 for cost effectiveness |
| Specialty inks | Metallic, puff, glow-in-dark for contemporary looks |

### Laser Engraving

| Material | Best Use |
|----------|----------|
| Wood | Warm, natural — best with Kufi or Thuluth |
| Acrylic | Modern, clean — backlit engrave for signage |
| Leather | Luxury — fine detail possible |
| Glass | Frosted effect — careful with heat stress |
| Metal | Durable — dark etch on silver/gold/black |

## Production-Ready Checklist

Before sending calligraphy artwork to production:

- [ ] Minimum stroke widths verified per production method
- [ ] Smallest legible size tested (print a sample at actual size)
- [ ] Color separation confirmed (spot colors, CMYK breakdown)
- [ ] Bleed applied (3mm minimum if touching edge)
- [ ] Outlines / vectorized (no live text or strokes)
- [ ] File format matches production method (CMYK EPS for print, SVG for digital)
- [ ] Pantone references provided for spot colors
- [ ] Embossing/foil die-line provided as separate layer
- [ ] Distortion applied if needed (curved surfaces, large formats)
- [ ] Proof approved at actual size and from actual viewing distance
