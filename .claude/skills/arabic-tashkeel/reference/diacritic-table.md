# Arabic Tashkeel — Complete Diacritic Reference

## Primary Diacritics (الحركات الأساسية)

| Mark | Name | Sound | Example | Unicode |
|------|------|-------|---------|---------|
| َ | Fatha (فتحة) | /a/ | دَ (da) | U+064E |
| ُ | Damma (ضمة) | /u/ | دُ (du) | U+064F |
| ِ | Kasra (كسرة) | /i/ | دِ (di) | U+0650 |
| ً | Fathatan (فتحتان) | /an/ | دً (dan) | U+064B |
| ٌ | Dammatan (ضمتان) | /un/ | دٌ (dun) | U+064C |
| ٍ | Kasratan (كسرتان) | /in/ | دٍ (din) | U+064D |
| ْ | Sukun (سكون) | No vowel | دْ (d) | U+0652 |
| ّ | Shadda (شدة) | Gemination | دَّ (dda) | U+0651 |
| ٰ | Superscript alif (ألف خنجرية) | Long /ā/ | اللّٰه | U+0670 |

## Automatic Tashkeel Rules (MSA)

| Rule | Example |
|------|---------|
| Definite article ال → al- | الْكِتَابُ |
| Hamzat al-wasl in command verbs | اِقْرَأْ (read!) |
| Prefixed b- takes kasra | بِالْكِتَابِ |
| Prefixed wa- takes fatha | وَالْكِتَابُ |
| Prefixed fa- takes fatha | فَالْكِتَابُ |
| Prefixed l- takes kasra | لِلْكِتَابِ |
