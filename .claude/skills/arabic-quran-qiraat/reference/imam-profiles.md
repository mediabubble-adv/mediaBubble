# Qira'at — Biographies of the Ten Imams

## The 7 Mutawatir Qira'at (7 أئمة القراءات السبع)

### 1. Imam Nafi' al-Madani (نافع المدني)
| Detail | Information |
|--------|-------------|
| Full name | Nafi' ibn 'Abd al-Rahman ibn Abi Nu'aym |
| Born | 70 AH / 689 CE (Medina) |
| Died | 169 AH / 785 CE (Medina) |
| Teachers | Abu Ja'far al-Madani, Yazid ibn al-Qa'qa' |
| Students | Qalun, Warsh (widely read in North/West Africa) |
| Method | Specialized in Madani transmission; chain to Ibn 'Abbas |
| Route use | **Warsh** — North/West Africa, parts of Sudan, Qatar; **Qalun** — Libya, Tunisia |

### 2. Imam Ibn Kathir al-Makki (ابن كثير المكي)
| Detail | Information |
|--------|-------------|
| Full name | 'Abdullah ibn Kathir al-Makki |
| Born | 45 AH / 665 CE (Mecca) |
| Died | 120 AH / 738 CE (Mecca) |
| Teachers | Abdullah ibn al-Sa'ib, Mujahid |
| Students | Al-Bazzi, Qunbul |
| Method | Meccan school; unique pronunciation of hamza |
| Route use | Less widely read today; preserved in Turkey and parts of Africa |

### 3. Imam 'Asim al-Kufi (عاصم الكوفي)
| Detail | Information |
|--------|-------------|
| Full name | 'Asim ibn Abi al-Najud al-Kufi |
| Born | 70 AH / 689 CE (Kufa) |
| Died | 127 AH / 745 CE (Kufa) |
| Teachers | Abu 'Abd al-Rahman al-Sulami, Zirr ibn Hubaysh |
| Students | **Hafs**, Shu'bah |
| Method | **Ḥafṣ 'an 'Āṣim** is the most widespread qira'a globally (~95% of printed Qurans) |
| Route use | **Hafs** — entire Muslim world (Egypt, Levant, Gulf, Asia); **Shu'bah** — parts of Sudan, West Africa |

### 4. Imam Hamzah al-Kufi (حمزة الكوفي)
| Detail | Information |
|--------|-------------|
| Full name | Hamzah ibn Habib al-Zayyat |
| Born | 80 AH / 699 CE (Kufa) |
| Died | 156 AH / 773 CE (Kufa) |
| Method | Known for lengthened madd (6 counts), tashil of hamzah |
| Important | His recitation is the most challenging due to heavy rules |
| Students | Khalaf (also one of the 10), Khallad |

### 5. Imam al-Kisa'i al-Kufi (الكسائي الكوفي)
| Detail | Information |
|--------|-------------|
| Full name | 'Ali ibn Hamzah al-Kisa'i |
| Born | 119 AH / 737 CE (Kufa) |
| Died | 189 AH / 805 CE (to Harun al-Rashid's camp near Rey, Iran) |
| Method | Close to Hamzah but lighter madd; famous as a grammarian |
| Students | Al-Layth ibn Khalid, Hafs ibn 'Umar al-Duri |

### 6. Imam Abu 'Amr al-Basri (أبو عمرو البصري)
| Detail | Information |
|--------|-------------|
| Full name | Zuban ibn al-'Ala' al-Tamimi |
| Born | 68 AH / 687 CE (Mecca, raised Basra) |
| Died | 154 AH / 771 CE (Kufa) |
| Method | Idgham kabir (large idgham) — merges similar letters extensively |
| Students | Al-Duri, al-Susi |

### 7. Imam Ibn 'Amir al-Shami (ابن عامر الشامي)
| Detail | Information |
|--------|-------------|
| Full name | 'Abdullah ibn 'Amir al-Yahsabi |
| Born | 21 AH / 642 CE (Damascus) |
| Died | 118 AH / 736 CE (Damascus) |
| Method | Levantine tradition; preserves certain long readings |
| Students | Hisham, Ibn Dhakwan |
| Route use | Historical Syrian region; still recited in parts of Levant |
