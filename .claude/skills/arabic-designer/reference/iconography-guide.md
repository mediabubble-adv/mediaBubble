# Arabic Design — Iconography & Symbolism Guide

## Icon Styles for Arab Audiences

| Style | Audience | Best For |
|-------|----------|----------|
| Flat/outline (minimal) | Youth, urban | Apps, modern brands |
| Detailed/ornamental | Conservative, traditional | Heritage, religious |
| Hand-drawn/informal | Social media, casual | Food, lifestyle |
| Geometric/patterned | Professional | Corporate, finance |

## Culturally Loaded Symbols

| Symbol | Meaning | Use / Avoid |
|--------|---------|-------------|
| 🌙 Crescent + star | Islam, spirituality | Widely positive; use for Ramadan, religious |
| 🏜 Desert + camel | Heritage, nostalgia | Acceptable but avoid cliché western tropes |
| 🏛 Column + arch | Classical Arab-Islamic civilization | Positive for heritage |
| 🕊 Olive branch + dove | Peace, Palestine | Sensitive — political nuance depends on context |
| ☪ Star (5-point) | Pan-Arab, Islam | Neutral — appears on many flags |
| 💧 Water drop | Life, Gulf heritage | Water scarcity symbolism is resonant |

## Colors in Icon Design

| Color | Meaning | Icon Application |
|-------|---------|-----------------|
| Gold | Luxury, faith, royalty | Accent icons, premium features |
| Teal | Professional, water, health | Healthcare, corporate |
| Terracotta | Earth, heritage, warmth | Cultural, tourism, food |
| Navy | Trust, technology | Banking, tech |
