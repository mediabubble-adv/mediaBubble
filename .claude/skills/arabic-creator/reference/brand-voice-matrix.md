# Brand Voice Matrix — R1/R2 Register Mapping to Masri

Maps MediaBubble's English brand registers (from `brand-voice-guidelines.md`) to Masri equivalents.

## Register Overview

| Voice | English Identity | Masri Identity | Tone |
|-------|-----------------|----------------|------|
| **R1 (Strategic Ally)** | Trusted partner, big-picture thinking, premium | شريك استراتيجي — واثق، محترف، دافئ | Professional, warm, authoritative |
| **R2 (Punchy Innovator)** | Energetic, tech-forward, bold | مبتكر جريء — سريع، تقني، جريء | Bold, energetic, direct |

## R1 — Strategic Ally (Masri)

**Register:** L3 (Polite Masri)
**Brand personality:** Confident, warm, expert

| English | Masri Equivalent | When to Use |
|---------|-----------------|-------------|
| "We understand..." | "احنا فاهمين..." | Empathy, consulting context |
| "Our approach..." | "طريقتنا في الشغل..." | Methodology explanation |
| "Let us help you..." | "خلينا نساعدك..." | Collaborative offer |
| "We've been doing this for..." | "احنا في المجال من..." | Credibility / longevity |
| "Our clients include..." | "من عملائنا..." | Social proof |
| "The result..." | "النتيجة؟..." | Before stating impact |
| "Imagine if..." | "تخيل لو..." | Vision-setting |

**Sentence style:** Longer, more structured, data-supported. Uses "نحن" for formal weight, "احنا" for warmth. Prefers full sentences over fragments.

**R1 Starter Phrases:**
- "في MediaBubble، بنشتغل على..." (At MediaBubble, we work on...)
- "من خبرتنا في السوق المصري..." (From our experience in the Egyptian market...)
- "هدفنا إننا نساعدك..." (Our goal is to help you...)
- "الفرق بين الطريقة التقليدية وطريقتنا..." (The difference between traditional and our method...)

## R2 — Punchy Innovator (Masri)

**Register:** L2–L3 (Daily to Polite Masri)
**Brand personality:** Energetic, tech-forward, bold

| English | Masri Equivalent | When to Use |
|---------|-----------------|-------------|
| "Stop wasting..." | "بطل تضيع..." | Pain-point opener |
| "Here's the thing..." | "خلينا نكون صريحين..." | Direct, no-fluff |
| "Get results fast" | "نتايج من أول شهر" | Benefit-driven |
| "AI-powered" | "بالذكاء الاصطناعي" | Differentiator |
| "While you sleep" | "وانت نايم" | Automation benefit |
| "Game changer" | "حاجة مختلفة خالص" | Innovation positioning |
| "No fluff" | "من غير لف ودوران" | Direct promise |

**Sentence style:** Shorter, punchier, question-driven. Starts with hooks and questions. Uses stronger verbs. Embraces more Masri colloquialisms (within L2 boundaries).

**R2 Starter Phrases:**
- "عاوز تعرف السر؟" (Want to know the secret?)
- "في حكاية..." (Here's the thing...)
- "تخيل إنك تقدر..." (Imagine you could...)
- "خلاص كده؟ لأ، فيه أكتر" (That's it? No, there's more)
- "النهارده هقولك..." (Today I'll tell you...)

## Forbidden Words Per Register

| Word | R1 | R2 | Notes |
|------|----|----|-------|
| يرجى | ✗ | ✗ | Too formal for both |
| معلش | ✓ (sparingly) | ✓ | R1: in responses only. R2: freely. |
| فشخ / جامد (slang) | ✗ | ✗ | Never. Use قوي / ممتاز instead. |
| حضرتك | ✓ | ✓ sparingly | R1: default. R2: only when addressing directly. |
| نحن | ✓ | ✗ | R1 for formal weight. R2 uses احنا. |
| إن شاء الله | ✓ | ✓ | Both: use naturally for future plans |
| ما شاء الله | ✓ | ✓ | Both: for praise/compliments |
