# YouTube SEO — Masri Video Content

## Arabic Title Optimization

### Best Practices
- **Length:** 40-60 characters (Arabic chars display wider)
- **Keyword placement:** Primary keyword in first 40 characters
- **Structure:** Main Keyword | Emotional Hook or Benefit
- **Masri:** Use Masri in titles (higher CTR from Egyptian viewers)
- **Numbers:** Include numbers when relevant (طريقتين, ٣ خطوات, ٥ أسباب)

### Title Templates

| Content Type | Template | Example |
|-------------|----------|---------|
| Educational | [Keyword] : [Benefit] | ألم الظهر : ٣ أسباب مش هتتخيلها |
| How-to | إزاي [Action] في [Context] | إزاي تختار دكتور أسنان في مصر |
| Testimonial | تجربتي مع [Provider/Doctor] | تجربتي مع عملية تجميل في مصر |
| Comparison | [Topic A] ولا [Topic B] : الفرق | الكشف أونلاين ولا في العيادة : مين الأفضل  |
| List | [Number] [Topic] في مصر | ٥ حاجات لازم تعرفها قبل عملية التجميل |
| Interview | [Name] : [Topic/Quote] | دكتور أحمد : إزاي نحافظ على صحة القلب |

### Title Checklist
- [ ] Under 60 characters
- [ ] Primary keyword in first 40 characters
- [ ] Masri dialect (not MSA formal)
- [ ] Compelling, not clickbait
- [ ] Matches video content (no misleading titles)
- [ ] Uses Arabic numbers where appropriate

## Arabic Tags Strategy

### Tag Hierarchy

```
Level 1 — Broad (1-2 tags):
صحة — طب — مصر

Level 2 — Niche (3-5 tags):
دكتور أسنان, جراحة تجميل, علاج الأسنان, صحة الفم

Level 3 — Specific (5-8 tags):
ألم ضرس, تركيب أسنان, دكتور أسنان في القاهرة, تبييض أسنان

Level 4 — Long-tail (3-5 tags):
إزاي اختار دكتور أسنان, أحسن دكتور أسنان في مصر, سعر تركيب الأسنان
```

### Tag Rules
- 10-15 tags per video
- Include primary keyword as first tag
- Include location tags for local targeting
- Include branded tags (MediaBubble)
- No tag stuffing (relevance matters more than quantity)
- Mix of MSA and Masri tags

## Arabic Subtitles & Captions

### Benefits
- Google indexes subtitle text for search
- Accessibility for hearing-impaired users
- Higher retention for Arabic viewers
- Google uses captions to understand video topic

### Implementation
- Upload Arabic SRT file (YouTube Studio → Subtitles → Upload)
- Use correct Arabic punctuation and spacing
- Match spoken Masri text (don't translate to MSA)
- Include sound effects and music descriptions in Arabic

### SRT Format Example

```
1
00:00:10,000 --> 00:00:15,000
اهلاً بيكم في قناة MediaBubble. النهاردة هنكلم عن حاجة مهمة.

2
00:00:15,000 --> 00:00:22,000
كتير مننا بيدور على دكتور مش عارف يختار منين. الفيديو ده هيساعدك.
```

## Thumbnail Text in Arabic

### Best Practices
- 3-5 words maximum (Arabic text needs space)
- High contrast (white or yellow text on dark background)
- Bold, sans-serif font (Cairo Bold or similar)
- Include keyword, question, or number
- Never cover the main focal point (face, product)

### Text Examples
| Video Content | Thumbnail Text (Arabic) |
|---------------|------------------------|
| How to choose a doctor | إزاي تختار؟ |
| Top 5 hospitals in Egypt | أحسن ٥ مستشفيات |
| Patient testimonial | تجربتي مع |
| Price comparison | الأسعار: مقارنة  |
| Health tip | معلومة مهمة |

## End Screen & Cards Strategy

### End Screen (Last 20 seconds)
- **Best for viewer:** Link to related video (Arabic content)
- **Subscribe:** Call to action in Arabic (اشترك في القناة)
- **Channel link:** Direct to MediaBubble channel

### Cards (During video)
- Place cards at natural transition points
- Link to related Arabic content (other videos, playlist)
- Link to website landing page for conversion-focused videos
- Use Arabic card title and CTA
