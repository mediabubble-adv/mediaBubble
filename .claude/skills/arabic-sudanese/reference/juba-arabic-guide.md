# Sudanese — Juba Arabic (عربية جوبا) Overview

## What Is Juba Arabic?

Juba Arabic is a **pidgin/creole** Arabic spoken in South Sudan (especially Juba) as a lingua franca. It is NOT standard Sudanese Arabic.

| Feature | Juba Arabic | Khartoum Arabic |
|---------|-------------|-----------------|
| Grammar | Simplified — no gender, no case, no verb conjugation by person | Full Arabic grammar |
| Vocabulary | ~70% Arabic + Nilotic substrate (Bari, Dinka) + English | Arabic (Sudanese variety) |
| Tense | Particles: **bi-** (present), **-ga-** (progressive), **-kan** (past) | Arabic verbal system |
| Gender | One form for all | Masculine / feminine |

## Core Grammar

| Feature | Juba Arabic | Equivalent in Sudanese |
|---------|-------------|----------------------|
| "I" | **ana** | أنا |
| "You" (sg) | **inta** (m) / **inti** (f) | إنتَ / إنتي |
| "He/she/it" | **hu** / **hi** / **de** | هو / هي |
| "We" | **anina** | نحن / حنا |
| "They" | **hinna** / **nasi** (**nass**) | هم |
| "To be" (present) | **fi** (existential) | في (k的存在) |
| Negation | **ma + verb** (no circumfix) | ما...ش |
| "Eat" | **akul(a)** (< ياكل) | ياكل |
| "Drink" | **jarib(a)** (< يشرب) | يشرب |

## Example Sentences

| Juba Arabic | Sudanese Arabic | English |
|-------------|-----------------|---------|
| ana mashi fi suuq | أنا ماشي في السوق | I'm going to the market |
| hu bi-giya | هو بيجي / جاي | He is coming |
| kalām da mā tamām | الكلام دا مش تمام | That word is not good |
| ajal ta'allim arabī | عشان تتعلم عربي | In order to learn Arabic |

## Relationship to MediaBubble

Juba Arabic is **not** the primary target for Egyptian/pan-Arab content. But if your content targets **South Sudanese diaspora** (Egypt, Uganda, Kenya) or **aid/NGO communications**, Juba Arabic is the correct register. Use standard Sudanese Arabic for Khartoum-focused content.
