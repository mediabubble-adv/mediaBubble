# Sudanese Arabic — Regional Dialect Guide

## Khartoum Arabic (Urban Standard)

| Feature | Detail |
|---------|--------|
| Qaf | q (preserved) or hamza (especially in rapid speech) |
| Jīm | ʒ (zh) — soft, like French "je" |
| Future | **حقـ (ḥag-)** — unique to Sudan |
| Negative | **ما + verb + ش** (mā ... sh) — like Maghrebi |
| Demonstratives | دا (m.), دي (f.), دول (pl.) |
| Key vocab | "لونك شنو؟" (how are you?), "تمّام" (fine) |

## Western Sudan (Darfur/Kordofan)

| Feature | Detail |
|---------|--------|
| Qaf | g (like Khaliji) |
| Future | **سا (sā)** / **حـ (ḥa-)** |
| Negative | ما + verb (without ش) |
| Substrate | Fur, Masalit, Zaghawa influences |
| Vocabulary | Unique terms for cattle, desert, tribal relations |

## Eastern Sudan (Kassala/Port Sudan)

| Feature | Detail |
|---------|--------|
| Future | راح (rāḥ) — shared with Beja |
| Substrate | Beja (Tu Bedawie) vocabulary |
| Qaf | Hamza (like Khartoum) |
| Maritime vocabulary | Red sea / port terms |

## Northern Sudan (Riverine Nile)

| Feature | Detail |
|---------|--------|
| Conservative | Preserves classical features |
| Greeting | Extended (5+ minutes) |
| Vocabulary | "جركوك" (girkok) — large water jug |
| Nubian influence | Some Nubian loanwords (kisra, etc.) |
