# Arabic Typography — Font Selection Guide

## Script Styles by Use Case

| Script | Usage | Readability | Best For |
|--------|-------|-------------|----------|
| Naskh | Body text, books, web | Highest | Long-form content, interfaces |
| Kufi | Headlines, display, branding | Moderate | Titles, logos, banners |
| Ruq'ah | Handwriting, informal notes | High | Casual, personal, short text |
| Thuluth | Ceremonial, titles | Low (ornamental) | Invitations, certificates |
| Diwani | Courtly, decorative | Low (conjoined) | Brand marks, luxury |
| Maghribi | North Africa | Moderate | Regional branding, Quran |

## Web Font Loading Strategy

| Strategy | FO*T/FOUT | When to Use |
|----------|-----------|-------------|
| `swap` (default) | Shows fallback then swaps | General web (creates FOUT) |
| `block` | Invisibly blocks | Brand-critical pages |
| `fallback` | Swap with short block | Balanced approach |
| `optional` | Swap only from cache | Performance-critical |
