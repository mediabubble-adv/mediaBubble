# Tajwid Rules — Complete Reference

## Makharij al-Huruf — 17 Articulation Points

### 1. Al-Jawf (The Throat Cavity) — 1 point
- Empty space of the mouth and throat
- Letters: Madd letters (ا و ي) when sakin
- Sound: natural prolongation

### 2. Al-Halq (The Throat) — 3 points

| Sub-point | Letters | Description |
|-----------|---------|-------------|
| Aqsa al-Halq (deepest) | ء (hamza), ه (ha) | Lowest part of throat |
| Wasat al-Halq (middle) | ع ('ayn), ح (ha) | Middle of throat |
| Adna al-Halq (closest) | غ (ghayn), خ (kha) | Closest to mouth |

### 3. Al-Lisan (The Tongue) — 10 points

| Sub-point | Letters | Description |
|-----------|---------|-------------|
| Back of tongue + soft palate | ق (qaf) | Deepest tongue point |
| Back of tongue + hard palate | ك (kaf) | Below qaf |
| Middle of tongue + palate | ج (jim), ش (shin), ي (ya) | Center of tongue |
| Side of tongue + molars | ض (dad) | Unique to Arabic |
| Tip of tongue + upper gum (slightly back) | ل (lam) | |
| Tip of tongue + lower teeth (with nasal) | ن (noon) | |
| Tip of tongue + upper gum (rolled) | ر (ra) | |
| Tip of tongue + base of front teeth | ط (ta), د (dal), ت (ta) | |
| Tip of tongue + edge of front teeth | ص (sad), ز (zay), س (sin) | |
| Tip of tongue + edge of front teeth | ظ (za), ذ (dhal), ث (tha) | |

### 4. Ash-Shafatan (The Lips) — 2 points

| Letters | Description |
|---------|-------------|
| ف (fa) | Inner lip + upper teeth |
| و (waw), ب (ba), م (meem) | Both lips |

### 5. Al-Khayshum (The Nasal Cavity) — 1 point
- Nasal passage
- Ghunnah: nasalization of noon and meem with shadda
- Duration: 2 harakat

## Sifaat al-Huruf — Letter Attributes

### Permanent Attributes (Sifat Lazima)

#### Opposing Pairs
| Pair | Attribute | Letters | Description |
|------|-----------|---------|-------------|
| Hams/Jahr | Hams (breath) | ف ح ث ه ش خ ص س ك ت | Breath flows |
| | Jahr (restraint) | Remaining | No breath flow |
| Shidda/Rakawa | Shidda (strong) | أ ج د ق ط ب ك ت | Complete stop |
| | Tawassut (medium) | ل ع م ن ر | Medium |
| | Rakawa (soft) | Remaining | Soft flow |
| Isti'la/Istifal | Isti'la (elevated) | خ ص ض غ ط ق ظ | Tongue raised |
| | Istifal (depressed) | Remaining | Tongue lowered |
| Idhqāq/Isfitāh | Idhqāq (narrow) | ل ر ن | Narrow passage |
| | Isfitāh (open) | Remaining | Open passage |

#### Non-Opposing Attributes
- Qalqala: vibration (ق ط ب ج د)
- Safir: whistling (ص ز س)
- Shadid: intensity
- Rakhwa: softness
- Mutawasit: medium

### Occasional Attributes (Sifat 'Arida)
- Tafkhim: heaviness
- Tarqiq: lightness
- Idgham: merging
- Ikhfa: concealment
- Iqlab: conversion
- Izhar: clarity

## Ahkam al-Madd — 12 Elongation Types

| Type | Arabic | Cause | Length | Example |
|------|--------|-------|--------|---------|
| Madd Tabi'i | طبيعي | Natural | 2 harakat | قالَ |
| Madd Wajib Muttasil | واجب متصل | Hamza after madd in same word | 4-5 harakat | جاءَ |
| Madd Ja'iz Munfasil | جائز منفصل | Hamza after madd in next word | 2-4-6 harakat | إِنَّا أَعْطَيْنَاكَ |
| Madd 'Arid li'l-Sukun | عارض للسكون | Stop after madd | 2-4-6 harakat | الْعَالَمِينَ (when stopping) |
| Madd Lazim Kalimi | لازم كلمي | Sukun after madd in word | 6 harakat | وَلَا الضَّالِّينَ |
| Madd Lazim Harfi Muthaqqal | لازم حرفي مثقل | Madd + shadda in letter | 6 harakat | المّ (alif-lam-meem with shadda) |
| Madd Lazim Harfi Mukhaffaf | لازم حرفي مخفف | Madd without shadda in letter | 2 or 6 harakat | نٓ |
| Madd Silah Sughra | صلة صغرى | Ha' damir between vowels | 2 harakat | لَهُ |
| Madd Silah Kubra | صلة كبرى | Ha' damir + hamza | 2-4-6 harakat | مَالَهُ أَخْلَدَهُ |
| Madd Farq | فرق | To differentiate | 6 harakat | آلذَّكَرَيْنِ |
| Madd Tamkin | تمكين | Two ya's together | 2 harakat | حُيِّيتُمْ |
| Madd Lazim | لازم | Necessary | 6 harakat | See above types |

## Waqf wa Ibtida — Pausing and Starting

### Types of Waqf
| Type | Arabic | Meaning |
|------|--------|---------|
| Waqf Lazim | لازم | Obligatory — meaning changes if continued |
| Waqf Ja'iz | جائز | Permissible — either pause or continue |
| Waqf Hasan | حسن | Good — pause is better |
| Waqf Qabih | قبيح | Bad — don't pause here |
| Waqf Daruri | ضروري | Necessary — due to breath |

### Mushaf Signs
| Sign | Meaning |
|------|---------|
| م | Waqf Lazim — must stop |
| لا | Do not stop |
| ج | Waqf Ja'iz — permissible |
| صلي | Better to continue |
| قف | Stop here |
| ∴ | Choose one of two places |
| ∴∴ | Stop at either, not both |

## Lahn — Errors

| Type | Arabic | Description | Ruling |
|------|--------|-------------|--------|
| Lahn Jali | جلي | Obvious error — changes meaning | Haram |
| Lahn Khafi | خفي | Hidden error — affects beauty | Makruh |

### Common Lahn Jali Examples
- Changing case endings (i'rab) that alter meaning
- Mispronouncing letters (ض as ظ, ذ as ز)
- Adding or dropping letters that change meaning
