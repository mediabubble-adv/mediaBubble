# arabic-msa — Modern Standard Arabic (Fus-ha)

## Overview

Modern Standard Arabic (MSA / الفصحى) is the formal, standardized register of Arabic derived from Classical Arabic. It is the language of formal writing, news broadcasting, official speeches, academic publishing, and literature across the entire Arab world. MSA is the lingua franca that connects Arabic speakers from Morocco to Iraq when their native dialects would not be mutually intelligible.

**Core principle:** MSA is not anyone's native language — it is learned through formal education. Mastery of MSA is the gateway to all other Arabic domains: dialectal, classical/religious, lexicographical, and literary.

## Core Grammar (Nahw)

### Parts of Speech Classification

| Category | Arabic Term | Description |
|----------|-------------|-------------|
| Noun | اسم | Subject, object, agent, instrument, place, time, etc. |
| Verb | فعل | Past (ماضي), present (مضارع), imperative (أمر) |
| Particle | حرف | Prepositions, conjunctions, interjections, negations |

### The Three Grammatical States (I'rab)

| State | Arabic | Markers | Function |
|-------|--------|---------|----------|
| Nominative | رفع (raf') | Damma / و / ا / ن | Subject, predicate, agent |
| Accusative | نصب (nasb) | Fatha / ي / ا / حذف ن | Object, circumstance, exception |
| Genitive | جر (jarr) | Kasra / ي | Preposition object, possession (idafa) |
| Jussive | جزم (jazm) | Sukun / حذف حرف علة / حذف ن | Conditional, prohibition |

## Morphology (Sarf)

### Root System Mastery

- Identify the triliteral (ثلاثي) or quadriliteral (رباعي) root of any Arabic word
- Distinguish between root letters (أصول) and augmentative letters (زوائد)
- Recognize the 10 augmented forms and their derivational families
- Apply rules of weak roots (مُعتلّ): assimilated (مثال), hollow (أجوف), defective (ناقص), doubly-weak (لفيف)
- Apply rules of geminate roots (مُضاعَف) and hamzated roots (مهموز)

### Morphological Scales (أوزان)

- Map every Arabic word to its morphological scale (وزن صرفي)
- Derive active participle (اسم فاعل), passive participle (اسم مفعول), noun of place/time (اسم مكان/زمان), noun of instrument (اسم آلة)
- Form diminutives (تصغير), relative adjectives (نسبة), and abstract nouns (مصدر صناعي)

## Rhetoric (Balaghah)

### Eloquence of Meaning (Ma'ani)

- Construct and identify declarative (خبري) and performative (إنشائي) sentences
- Apply emphasis techniques: qasam, innama, nafi + istithna', ta'kid
- Distinguish equative, restrictive, and topic-comment structures

### Eloquence of Expression (Bayan)

- Distinguish simile (tashbih), metaphor (isti'ara), and metonymy (kinayah)
- Classify metaphors into explicit (tasrihiyya), implicit (makniyya), and representative (tammithiliyya)
- Analyze how rhetorical figures operate differently in Arabic vs English

### Eloquence of Embellishment (Badi')

- Apply verbal embellishments: jinās (paronomasia), tibāq (antithesis), muqābala (contrast), tarsīʿ (rhymed prose), sajʿ (rhymed prose cadence)
- Apply semantic embellishments: tawriya (double entendre), murāʿāt al-naẓīr (congeniality), luzūm mā lā yalzam (obligatory non-obligation)

## Orthography (Imla')

- Correct hamza spelling rules: initial, medial, final, seat selection (alif, waw, yaa' seat)
- Hamzat al-wasl vs qat': identify and write correctly
- Taa' marbuta (ة) vs open taa' (ت): distinguish in writing and pronunciation
- Alif maqsura (ى) vs alif mamduda (ا): correct usage
- Shamsi vs qamari lam in definite article (ال): spelling and pronunciation distinction

## Composition (Insha')

- Write expository, argumentative, and narrative prose in formal MSA
- Apply proper paragraph structure with topic sentences and transition devices
- Maintain register consistency — no incursion of dialectal vocabulary or colloquial syntax
- Use appropriate formulaic expressions (افتتاحية، خاتمة، عبارات التعليل والاستدراك)

## Translation & Localization (English ↔ Arabic)

### Translation Philosophy

Arabic translation is not word-for-word mapping. Key principles:

| Principle | Explanation |
|-----------|-------------|
| **Functional equivalence** | Match the meaning and effect, not the words |
| **Register matching** | Map source register (casual/formal) to the correct Arabic variety |
| **Cultural adaptation** | Replace culture-specific references with Arabic-appropriate equivalents |
| **Directionality** | Arabic reads RTL; sentence restructuring is often needed, not just word swap |

### MSA vs Masri: When to Use Which

| Use MSA when | Use Masri when |
|--------------|----------------|
| Official documents, legal contracts | Marketing copy, social media, brand ads |
| News articles, press releases | Conversational chatbots, customer service |
| Academic content, research | Entertainment, comedy, informal content |
| Pan-Arab audience (no single dialect) | Audience is explicitly Egyptian |
| Religious or Quranic reference | Humor or emotional storytelling |

### Common Translation Errors to Avoid

| Error Type | Example | Fix |
|-----------|---------|-----|
| Literal loan | "كليك على الزر" (click on the button) | "اضغط على الزر" |
| False formal | Using فصحى for casual app UI | Use Masri or near-MSA |
| Gender mismatch | Masculine verb for female user | Match grammatical gender to user |
| Missing cultural anchor | Direct date reference without context | Add day + month + calendar context |
| Over-translating idioms | "Break a leg" → "اكسر رجلك" | Use Arabic equivalent: "بالتوفيق" |

### Terminology Reference by Field

| Field | Key Arabic Terms |
|-------|-----------------|

### Localization Checklist

- [ ] Date format: dd/mm/yyyy (not US mm/dd/yyyy)
- [ ] Currency: EGP (جنيه) or regional currency
- [ ] Calendar: Hijri dates where relevant (Ramadan, Islamic events)
- [ ] Phone numbers: +20 for Egypt, regional prefix
- [ ] Measurement: metric system (meters, kilograms)
- [ ] Cultural references: replace US/Western references with Arab equivalents
- [ ] Names: transliterate properly or use local name forms
- [ ] Numbers: Eastern Arabic numerals (١٢٣) in Arabic context vs. Western (123) in code/tech

### Agent Behavior Guidelines (Translation)

- **Always clarify target variety** (MSA or dialect) before translating
- **Explain translation decisions** — don't just output text, explain the choices
- **Flag cultural gaps** where direct translation loses meaning
- **Provide alternatives** when multiple valid translations exist
- **Note register shift** — if the source is casual, the translation should be too
- **Check terminology consistency** — same term should translate the same way throughout a document
