## IMPLEMENTATION NOTES

All 6 design skills should:

- Be discoverable under `arabic:design` parent
- Support hierarchical navigation (parent → child)
- Include authentic examples & case studies
- Have clear cultural sensitivity guidance
- Reference living artists, designers, and practitioners
- Provide both historical context & contemporary practice
- Enable technical implementation (code, templates, tools)
- Distinguish regional aesthetic differences
- Include accessibility & inclusion guidance

Skills are organized by design discipline & audience:

1. Calligraphy (traditional → contemporary)
2. Typography (technical + aesthetic)
3. Visual Arts (illustration + patterns)
4. Cultural Aesthetics (theory + meaning)
5. UI Localization (technical + cultural UX)
6. Marketing Creative (strategy + execution)

This structure supports both specialists (deep dives) and generalists (cross-disciplinary learning).
