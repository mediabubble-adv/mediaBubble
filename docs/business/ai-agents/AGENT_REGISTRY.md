## 🔧 Skill Definitions

### **Content Generation Skills**
- **Written Content:** Long-form copy, SEO-optimized articles, proposals, emails
- **Visual Content:** Design mockups, asset variations, graphics generation
- **Multimodal:** Video analysis, image processing, mixed media content

### **Analysis Skills**
- **Analytics:** Performance metrics, KPI tracking, trend identification
- **Data Analysis:** Lead scoring, financial data, market research
- **Market Research:** Competitor analysis, industry trends, opportunity identification

### **Optimization Skills**
- **SEO:** Keyword optimization, technical SEO, content SEO
- **A/B Testing:** Creative variations, copy testing, performance optimization
- **Performance:** Code optimization, speed improvement, efficiency gains

### **Communication Skills**
- **Real-time Response:** Instant replies, engagement handling, live chat
- **Personalization:** Tailored messaging, custom proposals, individual outreach
- **Brand Compliance:** Voice consistency, terminology adherence, style matching

### **Management Skills**
- **Planning:** Content calendars, project timelines, resource allocation
- **Prioritization:** Task scoring, backlog management, impact assessment
- **Reporting:** Dashboard creation, status reports, financial summaries
