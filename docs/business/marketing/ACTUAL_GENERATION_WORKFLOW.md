## STEP-BY-STEP WALKTHROUGH

### **Step 1: Go to Recraft**

1. Open https://app.recraft.ai
2. Click **"Continue as Guest"** (no signup needed)
3. You're in the editor

### **Step 2: Set Canvas**

- Select **"16:9 Widescreen"** on left panel
- Quality: **"Photorealistic"** or **"Mixed"**

### **Step 3: Generate v1**

1. Paste **Main Hero v1 prompt** into the text box
2. Click **"Generate"**
3. Wait ~15 seconds
4. When done, click the image to see full size
5. **Right-click → "Save Image As"**
6. Save as: `MainHero_v1.png`

### **Step 4: Generate v2, v3, v4**

Repeat Step 3 with each prompt (v2, v3, v4)

- Each takes ~15 seconds
- Total time: ~1 minute for 4 images

### **Step 5: Upload Back to Me**

Once you have 4 PNG files:

1. Tell me you're ready
2. I'll upscale all 4 to 4K with Magnific MCP
3. Done!
