## What We're NOT Changing

Your existing processes stay the same. The agents just pre-fill stuff for you:

- Sales still sends outreach emails (agent drafts them)
- Designers still make the final design (agent handles variations)
- Marketing still writes blogs (agent drafts them)
- Finance still approves invoices (agent generates them)
- Developers still merge PRs (agent just reviews faster)

**The agent output is optional but recommended.**

## FAQ

**Q: Will the agent replace my job?**
A: No. The agent replaces repetitive, low-value work (data entry, copy variations, code formatting). You focus on strategic/creative work.

**Q: What if the agent makes mistakes?**
A: You review before publishing/sending. The agent gets better with feedback.

**Q: Can I opt out?**
A: Yes, agents are optional. But we're measuring time savings across teams that use them.

**Q: What about privacy/data?**
A: Agent data stays within MediaBubble systems (HubSpot, GitHub, etc.). No external third parties.

**Q: What if I don't like a particular agent?**
A: Tell us. We can adjust, split, or retire it.

**Q: Can I customize the agent for my workflow?**
A: Phase 1 uses standard workflows. Phase 2 will have department-specific customization.
