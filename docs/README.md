## Folder Rules

- Keep active planning in the matching workstream folder.
- Move stale one-off material into [archive/](./archive/) instead of leaving duplicates in active folders.
- Prefer adding an index README in a folder before scattering more one-off notes inside it.
- Keep binary or source-reference files in [reference/](./reference/) so the active docs stay easy to scan.
