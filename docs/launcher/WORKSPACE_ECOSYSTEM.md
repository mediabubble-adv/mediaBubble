# MediaBubble Integrated Workspace Ecosystem

## Strategic Overview

The **Advanced Prompt Generator** serves as the **central asset hub** for all seven departments. This document outlines how each team uses, customizes, and amplifies the Prompt Generator within their specialized workflows.

**Architecture:** Single source of truth (Prompt Generator) → Distributed department specializations → Unified brand output

---

## 🎬 CONTENT CREATION WORKSPACE

### Mission
Transform brief ideas into ready-to-produce video/image briefs with complete creative direction.

### Integrated Workflow

```
Prompt Generator (Core)
    ↓
[Content Brief Builder]
    • Extract key elements from prompt chips
    • Auto-generate 5-point story structure
    • Suggest shot lists & transitions
    ↓
[Asset Library Integration]
    • Link to existing brand assets
    • Suggest stock footage/music tracks
    • Create mood boards from prompt
    ↓
[Production Checklist]
    • Equipment requirements
    • Crew roles
    • Timeline estimate
    ↓
[Export to Video Editor]
    • Send full brief to Premiere/DaVinci
    • Pre-cut timeline skeleton
    • Color grade presets loaded
```

### Key Features

**Visual Storyboarding**
- Prompt chips auto-map to shot types (Wide, Medium, Close-up)
- Lighting chip suggests key-fill-back ratios
- Camera chip generates technical specs sheet
- Safe zones chip creates graphics safe areas

**Content Calendar Integration**
- Generate 30-day content calendar from prompts
- Auto-schedule cross-platform formats
- Tag by content pillar (Product, Behind-the-Scenes, Educational, Social Proof)
- Batch export for scheduling tools (Buffer, Later, Hootsuite)

**Template System**
- Pre-built prompts for common formats (Product Demo, Testimonial, Tutorial, Promo)
- Quick-start modes (60-second TikTok, 2-minute explainer, 10-minute tutorial)
- Industry-specific templates (E-commerce, SaaS, Services, Agency)

### Mission

### Integrated Workflow

```

    ↓

    ↓
[Message Mapping]

    • Generate 3-5 campaign headlines
    • Create 10-point messaging pillar document
    ↓
[Asset Variations Generator]
    • Auto-create 10 variations of prompt
      (tone shift, angle shift, emphasis shift)
    • Generate copy for each variation
    ↓
[Channel Distribution]

    • Platform-specific copy (150 chars for Twitter, etc.)
    • Posting schedule optimization
    ↓
[Performance Tracking]

```

### Key Features

**Campaign Brief Generator**
- Input campaign goal → Auto-suggest campaign name, duration, budget allocation
- Prompt directly maps to audience persona
- Auto-generate target KPIs (Impressions, CTR, Conversions)
- Create campaign timeline with milestones

**Messaging & Copy Framework**
- Core message extraction (1 sentence)
- Key message pillars (3-5 main points)
- Proof points (Social proof, data, testimonials)
- Objection handlers (Why now? Why us? What's different?)

**Channel-Specific Optimization**
- Facebook/Instagram: 3 headline versions, 5 description options
- LinkedIn: Executive summary + 3-point benefit breakdown
- Email: Subject line (20 variations), preview text, body copy
- Twitter/X: Tweet thread + 10 standalone tweets
- TikTok: Hook (first 3 seconds), body, outro formula

**Design System Auto-Generation**

- Component library structure (foundation → compound → templates)

**Component Specification**
- Button variants (primary, secondary, ghost, loading states)
- Form inputs (text, email, phone, select, checkbox, radio, toggle)
- Cards & modals (header, body, footer, overlay specs)
- Navigation (header, sidebar, breadcrumbs, tabs, pagination)
- Data display (tables, lists, charts, metrics)

**Interaction Framework**
- Hover states (color shift, shadow lift, scale)
- Focus states (ring + offset for accessibility)
- Active states (background fill, pressed effect)
- Loading states (skeleton, spinner, progress bar)
- Empty states (illustration + helpful text)
- Error states (validation, inline error, toast)
