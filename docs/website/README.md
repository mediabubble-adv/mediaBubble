## THE SERVICE PAGE SYSTEM

### Why This Matters

You have 15+ services across 4 pillars. Currently:

- ❌ Each page built differently (2-4 hours each)
- ❌ Inconsistent design & messaging
- ❌ Hard to update brand colors
- ❌ No clear conversion structure

### The Solution: Template + Components

```
Service Page = Hero + [Flexible Sections] + CTAs
```

**11 Section Types (Pick & Order):**

1. Hero / Page Title (required)
2. Problem Statement (required)
3. How It Works / Process (required)
4. Features / Benefits Grid (required)
5. Results / ROI Section
6. Case Study / Social Proof (required)
7. Testimonials (optional)
8. FAQ Accordion (optional)
9. Pricing (optional)
10. Tools & Technologies (optional)
11. Related Services Grid (required)

**10+ Reusable Components:**

- Buttons (6 variants)
- Cards (4 types)
- Sections
- Headers
- Testimonials
- FAQ
- Pricing tables
- Stats boxes
- Timeline
- Icon grids

**Result:**

- New service page in 30-45 minutes (was 4-6 hours)
- 100% consistent styling
- 3 strategically-placed CTAs per page
- Easy A/B testing
- Scalable for future services

## THE CONVERSION ARCHITECTURE

### Strategic CTA Placement

Every service page has **3 conversion moments:**

```
CTA #1: HERO
├─ Placement: Top of page
├─ Copy: "Get Free Consultation"
├─ Goal: Early engagement for hot leads
└─ Target: Decision-makers ready to act

CTA #2: MID-PAGE
├─ Placement: After social proof (case study)
├─ Copy: "Schedule Consultation"
├─ Goal: Convert after trust is built
└─ Target: Leads convinced by evidence

CTA #3: FOOTER
├─ Placement: Bottom of page
├─ Copy: "Start Your Project"
├─ Goal: Last chance for committed leads
└─ Target: Ready to move forward
```

**Plus:**

- Newsletter popup (exit-intent)
- Floating CTA (always visible)
- Related services cross-sell

## THE TECHNICAL FOUNDATION

### Component System

**CSS Variable System (Design Tokens):**

```css
Colors:
--brand-primary: #0D3A7D
--brand-secondary: #FFC107
--brand-accent: #2196F3

Spacing (8px base):
--space-md: 16px
--space-lg: 24px
--space-xl: 32px

Typography:
--font-size-base: 16px
--font-weight-semibold: 600
--font-weight-bold: 700

Transitions:
--transition-normal: 300ms cubic-bezier(...)
--transition-slow: 500ms cubic-bezier(...)

Shadows & Radius:
--shadow-lg: 0 10px 20px rgba(...)
--radius-lg: 12px
```

**Component Library Structure:**

```
Components/
├─ Buttons (primary, secondary, outline, ghost, lg, sm)
├─ Cards (service, feature, testimonial, case-study)
├─ Sections (hero, problem, features, testimonials, cta)
├─ Forms (input, textarea, select, validation)
├─ Icons (grid, single, with-label)
├─ Timeline (process steps)
├─ Pricing (tables, comparison, cards)
├─ Stats (boxes, metrics)
└─ FAQ (accordion)
```
