## How To Use This Folder

1. Start with the improvement plan to understand the active priorities.
2. Use the architecture guide when changing page composition or reusable sections.
3. Use the comparison doc when you need the design intent behind a UI change.
4. Keep new site work here instead of scattering it into `docs/` root folders.
