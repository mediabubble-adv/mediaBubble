# Reference Material

This folder collects source files, legacy exports, SOPs, and other material that informs the active docs and codebase.
It is a reference shelf, not a place for active planning.

## What Is Here

| Folder | Contents |
| --- | --- |
| [01-Branding/](./01-Branding/) | Original brand source files |
| [AI-Content/](./AI-Content/) | Draft website content and document exports |
| [SOPs/](./SOPs/) | Operations spreadsheets and process references |
| [mediabubble-scrape/](./mediabubble-scrape/) | Legacy scrape utilities and sitemap notes |

## How To Use It

1. Pull facts or source material from here when you need historical context.
2. Copy useful material into the active docs folders if it becomes part of an ongoing workflow.
3. Do not keep duplicate strategy notes here once they belong in `docs/brand/`, `docs/website/`, or another active folder.
