## The Master Agent Innovation

**Key insight:** Instead of thinking "consolidate by department," think "consolidate by business outcome."

Each master agent is **configuration-driven**, not **logic-duplicating**:

```
BEFORE (scattered):
- Social Content Agent
- Blog Generator Agent
- Video Caption Agent
- Email Copy Agent
- Ad Copy Agent
→ 5 separate agents, 5 separate implementations

AFTER (master agents):
- Content & Copy Engine
  with configuration:
  { type: "social", platform: "instagram", ... }
  { type: "blog", subject: "seo", ... }
  { type: "video", platform: "tiktok", ... }
  { type: "email", subject: "campaign", ... }
  { type: "ad", platform: "google", ... }
→ 1 agent, infinite configurations
```

This is the principle behind all 6 master agents.
