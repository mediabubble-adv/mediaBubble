# Skill Power Levels — From Basic to Visionary

---

## TIER 1: FOUNDATION SKILLS ⭐

**Examples:** Syntax checker, color validator, tone detector

```
User: "Check if this color meets accessibility standards"

Skill Response:
  ✓ WCAG AA compliant
  (Done)
```

**What it does:** Single task, no reasoning, no context  
**Power:** Basic utility  
**Useful for:** Quick validation  
**Learning:** None

**Examples:** Arabic Language Master, Design System Master, Brand Voice Enforcer

```

  2. Why this works for this audience
  3. Predicted engagement (87%)

  5. Cultural warnings (important)
  6. Code-switch recommendations
```

**What it does:** Deep domain expertise + reasoning + context awareness  
**Power:** Expert consulting

**Learning:** Yes (remembers patterns)
