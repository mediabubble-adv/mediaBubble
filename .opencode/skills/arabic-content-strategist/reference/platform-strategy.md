# Platform Strategy — Egyptian Audience

## Instagram

| Aspect | Strategy |
|--------|----------|
| **Primary audience** | Health-conscious consumers (22-40), medical tourists |
| **Content mix** | 50% Reels, 30% carousels, 10% single image, 10% Stories |
| **Peak time** | 9-11 PM (Ramadan: 11 PM-1 AM), 2-4 PM weekday |
| **Tone** | R2 (Punchy Innovator) — visual-first, minimal text, bold captions |
| **Best formats** | Before/after transformations, quick tips, patient journey reels, doctor introductions |
| **Hashtag strategy** | Mix of broad (مصر, صحة) + niche (دكتور_تجميل, سياحة_علاجية) + branded (MediaBubble) |
| **Masri hooks** | "بصوا كده", "مصدقتش", "الحق نفسك", "قولي وداعاً لـ" |
| **CTAs** | الرابط في البايو, احجز دلوقتي, اعرف أكتر |
| **Frequency** | 5-7 posts/week + daily Stories |
| **Story format** | Polls about health concerns, Q&A stickers, countdown to events, swipe-up to booking |

### Content Calendar per Week (Instagram)

| Day | Format | Content | CTA |
|-----|--------|---------|-----|
| Sat | Reel | Educational tip (30-60s) | اعرف أكتر |
| Sun | Carousel | Step-by-step guide | احفظ البوست |
| Tue | Reel | Patient story / testimonial | احجز دلوقتي |
| Wed | Carousel | Comparison / myth vs fact | شارك البوست |
| Thu | Reel | Weekend health challenge | جرب بنفسك |

## TikTok

| Aspect | Strategy |
|--------|----------|
| **Primary audience** | Younger consumers (18-30), health-curious |
| **Content mix** | 100% short-form video (15-60s) |
| **Peak time** | 7-10 PM, 12-2 PM |
| **Tone** | R2 (Punchy Innovator) — ultra-casual, trending sounds, fast pace |
| **Best formats** | POV clips, myth busting, "day in the life" of doctor, quick tips, trending audio duets |
| **Hashtag strategy** | Egyptian trends (#مصر, #معلومة_طبية), niche health (#نصيحة_دكتور) |
| **Masri hooks** | "حاجة هتغير حياتك", "قولي..", "بصراحة", "أنا صدمت" |
| **CTAs** | تابعني عشان, احفظ الفيديو, علق لنا |
| **Frequency** | 3-5 posts/week |
| **Specs** | Vertical 9:16, highly captioned, fast cuts, text overlay |

## Facebook

| Aspect | Strategy |
|--------|----------|
| **Primary audience** | Broader age range (25-55), professional + personal, secondary = medical professionals |
| **Content mix** | 40% posts, 25% carousels, 20% articles, 15% videos |
| **Peak time** | 8-11 PM, Friday 11 AM-1 PM |
| **Tone** | R2 (Punchy Innovator) with R1 (Strategic Ally) for articles |
| **Best formats** | Health tips, shared articles, polls, community discussions, event promotion, longer videos |
| **Groups strategy** | Join Egyptian health groups, share value without hard selling |
| **Masri hooks** | "عايزين رأيكم", "إيه رأيك في ..", "نقاش مهم", "معلومة جديدة" |
| **CTAs** | اكتب رأيك, شارك البوست, تابع صفحتنا |
| **Frequency** | 5-7 posts/week |
| **Special** | Best platform for long-form Masri posts (500-1000 chars) |

## LinkedIn

| Aspect | Strategy |
|--------|----------|
| **Primary audience** | Healthcare decision-makers, professionals, investors |
| **Content mix** | 40% articles, 30% short posts, 20% carousels, 10% documents/PDFs |
| **Peak time** | 8-9 AM, 12-1 PM, 5-6 PM (Sunday-Thursday) |
| **Tone** | R1 (Strategic Ally) — professional, data-driven, authoritative |
| **Best formats** | Industry analysis, case studies, thought leadership, research summaries, company milestones |
| **Masri/MSA balance** | 60% MSA + 40% Masri (Masri for relatable examples and quotes) |
| **CTAs** | تواصل معنا, حمل التقرير, شارك رأيك, احجز استشارة |
| **Frequency** | 3-4 posts/week |
| **Document sharing** | Upload PDF reports, whitepapers, infographics as document posts |

## YouTube

| Aspect | Strategy |
|--------|----------|
| **Primary audience** | Health information seekers, medical tourists, professionals |
| **Content mix** | 50% educational, 25% testimonials, 15% interviews, 10% virtual tours |
| **Peak time** | 7-11 PM, Friday afternoon |
| **Tone** | R1 (Strategic Ally) — in-depth, credible, visually clean |
| **Best formats** | Doctor interviews, clinic tours, procedure explainers, patient journeys |
| **Masri approach** | Spoken Masri with subtitles (Arabic + English) |
| **SEO** | Arabic keyword-optimized titles with English translation in description |
| **CTAs** | اشترك في القناة, تواصل معنا, احجز استشارة مجانية |
| **Frequency** | 1-2 videos/week |
| **Length** | 5-15 minutes (educational), 2-5 minutes (testimonials) |

## Website / Blog

| Aspect | Strategy |
|--------|----------|
| **Primary audience** | All segments (search-driven) |
| **Content mix** | Guides, landing pages, articles, provider directories, comparison pages |
| **SEO priority** | Masri keywords + MSA long-tail healthcare searches |
| **Tone** | R1 (Strategic Ally) — comprehensive, trustworthy, authoritative |
| **RTL** | Full RTL layout with Cairo font at configured weights |
| **Best formats** | Ultimate guides, condition explainers, provider comparisons, insurance guides |
| **CTAs** | احجز موعد, اعرف أكتر, قارن بين, تواصل معنا |
| **Frequency** | 4-8 new pages/month |
| **Function** | Hub for ALL platform content, longest shelf life |
