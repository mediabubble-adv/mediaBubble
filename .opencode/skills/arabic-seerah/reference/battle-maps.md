# Seerah — Key Battles Geography & Strategy Reference

## Badr (17 Ramadan 2 AH / 17 March 624)

| Element | Detail |
|---------|--------|
| Location | Badr, 130 km SW of Medina — well-watered valley |
| Forces | Muslims: 313 (70 Muhajirun, 236 Ansar, 7 camels, 3 horses); Quraysh: ~1,000 (200 horses, 700 camels) |
| Muslim Strategy | Block water access → camp on elevated ground → controlled morning sun |
| Key Moment | "سيهزم الجمع ويولون الدبر" — angelic reinforcement mentioned in Quran 3:123–125 |
| Casualties | Muslims: 14 martyrs; Quraysh: 70 killed, 70 captured |
| Significance | First battle; established Muslim military credibility; raised morale |

## Uhud (7 Shawwal 3 AH / 23 March 625)

| Element | Detail |
|---------|--------|
| Location | Mount Uhud, 5 km N of Medina |
| Forces | Muslims: ~700 (300 were munafiqun who retreated under Ibn Salul); Quraysh: ~3,000 (200 cavalry, led by Abu Sufyan) |
| Strategy | 50 archers on Mount 'Aynayn to protect flank |
| **Turning point** | Archers left post for spoils → Khalid ibn al-Walid (not yet Muslim) flanked them |
| Result | Muslims suffered heavy losses (70 martyrs including Hamzah) |
| Lesson | Obedience to command; discipline over material gain |
| Quran | Surah Al 'Imran (3):121–179 covers the battle in detail |

## Khandaq (Battle of the Trench) — Shawwal 5 AH / April 627

| Element | Detail |
|---------|--------|
| Location | Medina — defensive trench dug on the northern approach |
| Forces | Muslims: ~3,000; Confederates: 10,000+ (Quraysh, Ghatafan, Banu Nadir, etc.) |
| Strategy | Salman al-Farisi's trench idea (Persian defensive technique) |
| Duration | 15–30 days of siege |
| Key event | Nu'aym ibn Mas'ud spreads discord among Confederates; violent windstorm at night |
| Result | Confederates retreat without a pitched battle |
| Quran | Surah al-Ahzab (33):9–27 |
| Significance | Last major non-Muslim military threat against Medina |

## Khaybar (Muharram 7 AH / May 628)

| Element | Detail |
|---------|--------|
| Location | Khaybar oasis, ~150 km N of Medina |
| Forces | Muslims: ~1,600; Jewish tribes of Khaybar: 10,000+ in fortresses |
| Strategy | Siege of 8 fortresses (al-Na'im, al-Qamus, etc.) over ~30 days |
| Key moment | Ali breaks through fortress gate (single-handed, tradition says) |
| Result | Jewish surrender; peace terms — they remain on land, give 50% produce |
| Effect | Secures northern trade route; Jewish tribes no longer a military threat |

## Conquest of Mecca (20 Ramadan 8 AH / 11 January 630)

| Element | Detail |
|---------|--------|
| Forces | Muslims: 10,000 (largest Muslim army yet) |
| Advance | 4 columns enter Mecca from 4 directions |
| Resistance | Almost none — Quraysh surrendered; only Khalid's column faced minor skirmish |
| **Prophet's decree** | General amnesty — "اذهبوا فأنتم الطلقاء" (Go, you are free) |
| Result | Kaaba cleansed of idols; Quraysh nobility converts en masse |
