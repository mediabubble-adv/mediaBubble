# Arabic Lexicography — Al-Furuq al-Lughawiyyah (Subtle Semantic Differences)

## Near-Synonym Differentiation Index

### Basic Concepts

| Pair | Meaning | Difference |
|------|---------|------------|
| قعد / جلس | Both "to sit" | قعد = to rise from standing → to sit; جلس = to sit from lying down (also: "to sit" as in sitting for a purpose) |
| كرم / شرف | Both "noble" | كرم = innate generosity; شرف = inherited/honorary status |
| شجاعة / جرأة | Both "bravery" | شجاعة = bravery with thought; جرأة = reckless/daring |
| علم / عرف | Both "to know" | علم = to know something unknown; عرف = to recognize something already known |

### Emotion & Disposition

| Pair | Difference |
|------|------------|
| حب / عشق | حب = general love; عشق = excessive love that overpowers |
| غضب / سخط | غضب = anger; سخط = anger with disapproval |
| حزن / كآبة | حزن = manageable sadness; كآبة = depression (from hidden cause) |
| خوف / خشية | خوف = fear of harmful thing; خشية = reverential fear (God, authority) |
| قلق / اضطراب | قلق = mental unease; اضطراب = physical + mental agitation |

### Action & Movement

| Pair | Difference |
|------|------------|
| مضى / ذهب | مضى = to go forward (linear); ذهب = to go away (dispersal) |
| جاء / أتى | جاء = to come (common); أتى = to come (with purpose/difficulty context) |
| سار / مشى | سار = to travel/walk (general); مشى = to walk (pedestrian) |
| رمى / طرح | رمى = to throw (with force); طرح = to cast aside |
| أخذ / نال | أخذ = to take; نال = to obtain/succeed in taking |

### Quantity & Degree

| Pair | Difference |
|------|------------|
| كثير / وفير | كثير = abundant (count); وفير = abundant (quality/naturalness) |
| قليل / نادر | قليل = few in count; نادر = scarce/rare |
| شديد / عنيف | شديد = intense (general); عنيف = violent (implies harm) |
