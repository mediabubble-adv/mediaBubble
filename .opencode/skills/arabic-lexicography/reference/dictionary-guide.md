# Arabic Lexicography — Classical Dictionary Navigation Guide

## The Major Classical Dictionaries

### 1. العين (Al-ʿAyn) — First Arabic Dictionary

| Detail | Value |
|--------|-------|
| Author | Al-Khalil ibn Ahmad al-Farahidi |
| Died | 175 AH / 791 CE |
| Organization | **Phonetic** — ordered by articulation point (throat → lips) |
| Structure | Al-Kitab (the book) organized by root; each letter section = Kitāb |
| Strength | First dictionary; best for etymology and early usage |
| Weakness | Missing some later words; idiosyncratic organization |
| Finding a word | Know the root; check ع → closest to throat |

### 2. جمهرة اللغة (Jamharat al-Lughah)

| Detail | Value |
|--------|-------|
| Author | Ibn Duraid |
| Died | 321 AH / 933 CE |
| Organization | Phonetic by final consonant (reverse of al-ʿAyn) |
| Strength | Easy lookup for rhyme; focuses on authentic Arabic |
| Unique | Each letter chapter has "Nawadir" (rare words) section |

### 3. تهذيب اللغة (Tahdhib al-Lughah)

| Detail | Value |
|--------|-------|
| Author | al-Azhari |
| Died | 370 AH / 980 CE |
| Organization | Mixed: follows al-ʿAyn's partial system + replaces problematic entries |
| Strength | Corrects al-ʿAyn; includes Bedouin attestations |

### 4. المحيط في اللغة (al-Muhit fi al-Lughah)

| Detail | Value |
|--------|-------|
| Author | al-Sahib ibn 'Abbad |
| Died | 385 AH / 995 CE |
| Organization | Phonetic like al-ʿAyn |
| Strength | Comprehensive; one of the largest pre-modern dictionaries |

### 5. الصحاح (al-Sihah) — Most Popular Classical

| Detail | Value |
|--------|-------|
| Author | al-Jawhari |
| Died | 400 AH / 1009 CE |
| Organization | **Rhyme-based** (by final consonant, common in Arabic lexicography) |
| Name | "الصحاح" = "The Sound [Words]" |
| Strength | Most reliable single-volume dictionary; Hadith-friendly |
| Weakness | Missing some words (al-Saghani later wrote "التكملة" = The Completion) |

### 6. لسان العرب (Lisan al-Arab) — The Encyclopedia

| Detail | Value |
|--------|-------|
| Author | Ibn Manzur |
| Died | 711 AH / 1311 CE |
| Organization | Alphabetical by root (modern system) |
| Content | Compilation of 5 earlier works (Tahdhib, Sihah, etc.) |
| Volumes | ~20 (print) / 18 (standard edition) |
| Strength | Complete medieval Arabic vocabulary; citations and attestations |
| Weakness | Length; no definitions for technical/philosophical terms |

### 7. القاموس المحيط (al-Qamus al-Muhit)

| Detail | Value |
|--------|-------|
| Author | al-Firuzabadi |
| Died | 817 AH / 1415 CE |
| Organization | Alphabetical by root (refined Ibn Manzur) |
| Strength | Concise definitions; "the Ocean" — manageable 4-volume work |
| Weakness | Overly concise; loses contextual nuance |

### 8. تاج العروس (Taj al-'Arus) — Final Classical

| Detail | Value |
|--------|-------|
| Author | al-Murtada al-Zabidi |
| Died | 1205 AH / 1791 CE |
| Organization | Commentary on al-Qamus al-Muhit |
| Strength | Definitive classical lexicon; indexes all earlier works |
| Length | ~40 volumes |
