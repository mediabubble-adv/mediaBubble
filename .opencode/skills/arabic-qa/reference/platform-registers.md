# Platform Register Reference

Target register and content rules per platform.

## Social Media

### Facebook

| Field | Value |
|-------|-------|
| **Default register** | L2–L3 (Daily to Polite Masri) |
| **Why** | Mixed-age audience (25–55). Broadest reach. Needs warmth for engagement, polish for credibility. |
| **Character limit** | 63,206 (post) / 125 (headline) / 30 (CTA button) |
| **Optimal length** | 80–150 characters for ads, 150–300 for organic posts |
| **Hashtag limit** | 3–5 hashtags max |
| **Emoji use** | 0–2 per post. Use to soften tone or add personality, not decoration. |
| **CTA style** | Direct but polite: "سجل بياناتك واحنا نكلمك" |
| **Best posting time** | 8–10 AM, 2–4 PM, 8–11 PM Egypt time |
| **Notes** | Facebook audience skews older than Instagram — avoid L1 slang even in comments |

### Instagram

| Field | Value |
|-------|-------|
| **Default register** | L2 (Daily Masri) |
| **Why** | Visual-first, younger skew (18–40). Higher energy, shorter attention. |
| **Character limit** | 2,200 (caption) / 150 (first 2 lines visible) |
| **Optimal length** | 60–125 characters for posts, 125–200 for carousel |
| **Hashtag limit** | 5–10 hashtags (mix of broad, mid, niche) |
| **Emoji use** | 1–2 per caption. 0 for carousel educational content. |
| **CTA style** | Conversational: "إيه رأيك؟" / "قولنا في كومنت" |
| **Best posting time** | 9–11 AM, 7–10 PM Egypt time |
| **Stories register** | L2 (or L1 for casual content, polls, Q&A) |
| **Notes** | First 2 lines are critical — the "more" cutoff. Hook in first line. |

### LinkedIn

| Field | Value |
|-------|-------|
| **Default register** | L3–L4 (Polite to Formal Masri) |
| **Why** | Professional audience, decision-makers. Needs data, proof, measured tone. |
| **Character limit** | 3,000 (post preview) / 125 (headline) |
| **Optimal length** | 150–300 characters for posts, 500–1500 for articles |
| **Hashtag limit** | 3–5 hashtags max |
| **Emoji use** | 1 in post preview max. Zero in articles. |
| **CTA style** | Professional: "للتواصل، راسلنا على..." / "احجز استشارة" |
| **Best posting time** | 7–9 AM, 12–1 PM, 5–6 PM Egypt time (Sun–Thu) |
| **Notes** | Masri on LinkedIn is a differentiator. Most Egyptian companies use MSA here. Masri L3 reads as approachable expert, not unprofessional. |

### TikTok

| Field | Value |
|-------|-------|
| **Default register** | L1–L2 (Street to Daily Masri) |
| **Why** | Youth (15–30), entertainment-first, trend-driven. Speed and energy matter. |
| **Video length** | 15–60 seconds |
| **Text overlay** | L2 Masri, 3–5 words per frame |
| **Caption limit** | 150 characters (preview) / 2,200 (full) |
| **Hashtag limit** | 2–3 trending Egyptian hashtags + 2 broad |
| **Emoji use** | 1–3 per caption. High energy. 😂 and 💀 are native TikTok vocab. |
| **Hook rule** | First 2 words must grab attention: "قولي بقى..." / "تخيل..." / "بصوا..." |
| **Best posting time** | 6–9 PM, 11 PM–1 AM Egypt time |
| **Notes** | TikTok Masri should feel like a friend talking, not a brand. Imperfect = authentic. |

## Direct Communication

### Email

| Field | Value |
|-------|-------|
| **Default register** | L3 (Polite Masri) |
| **Why** | Direct-to-inbox, mixed audience. Needs structure + warmth. |
| **Subject line length** | 30–50 characters |
| **Preheader length** | 80–100 characters |
| **Body length** | 100–300 words |
| **Emoji in subject** | Use sparingly — only for promo/newsletter, not for formal outreach |
| **Emoji in body** | 0–2 per email |
| **CTA style** | Single, clear CTA button + optional text link |
| **Structure** | Subject → Greeting (أهلاً [name]) → Body (Masri L3, scannable) → CTA → Signature |
| **Best sending time** | 8–10 AM, 2–4 PM Egypt time (Sun–Thu) |
| **Notes** | Avoid L1 entirely in email. L3 strikes the right balance of personal + professional. |

### WhatsApp

| Field | Value |
|-------|-------|
| **Default register** | L2–L3 (Daily to Polite Masri) |
| **Why** | Personal channel, conversational. One-to-one or small group. |
| **Length** | 30–100 words per message |
| **Emoji use** | 1–2 per message. 👋 is expected for opener. |
| **CTA style** | Soft, low-pressure: "ممكن نتكلم؟" / "حابب نكمل؟" |
| **Greeting** | "السلام عليكم [name] 👋" for new contacts. "أهلاً [name] 👋" for warm leads. |
| **Notes** | WhatsApp is intimate. Don't blast. Don't use bullets or formatted lists. Keep it to one topic per message. |

## Web

### Landing Page

| Field | Value |
|-------|-------|
| **Default register** | L3 (Polite Masri) |
| **Why** | First impression, credibility signal. Needs to earn trust fast. |
| **Hero headline** | 8–12 words. MSA or Masri L3. Authority first. |
| **Subheadline** | 12–20 words. Masri L3. Clarity + warmth. |
| **Body** | Masri L3 with proof points (numbers, testimonials, logos). |
| **CTA button** | Masri L2–L3. Action-oriented but not pushy: "سجل واحنا نكلمك" |
| **Testimonials** | Keep customer quotes exactly as they said them (register varies naturally). |
| **Legal / Footer** | MSA (perceived legitimacy for fine print). |
| **Optimal length** | 500–1000 words per section. Scannable with Masri subheadlines. |
| **Notes** | First page view is 5–10 seconds. Hook + proof + CTA must fit above the fold. |

### Blog

| Field | Value |
|-------|-------|
| **Default register** | Mixed: MSA H1 + Masri L3 body |
| **Why** | SEO: Google indexes MSA headlines better. Engagement: Masri body keeps readers 2x longer. |
| **H1** | MSA — targets informational search queries |
| **H2s** | Masri L3 — conversational, engaging subheadlines |
| **Body** | Masri L3 — readable, personal, keeps dwell time high |
| **FAQ** | Masri L2 — actual Egyptian phrasing from People Also Ask |
| **Meta description** | MSA + key terms |
| **Optimal length** | 1000–2000 words |
| **Notes** | The mixed MSA/Masri approach gives you the best of both worlds: search indexing + reader retention. |
