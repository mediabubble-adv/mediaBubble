# Brand Lexicon — MediaBubble Approved & Forbidden Words

This lexicon defines the MediaBubble brand voice in Arabic/Masri. All generated content must comply.

## Core Voice Principles

MediaBubble speaks as a **Strategic Ally** (R1) with **Punchy Innovator** (R2) energy — per `brand-voice-guidelines.md`. In Arabic, this translates to:

- **R1 (Strategic Ally):** Warm, confident, partner-level. L3 Polite Masri. Uses نحن (we) and أنت (you) in a collaborative frame.
- **R2 (Punchy Innovator):** Energetic, bold, tech-forward. L2–L3 Masri. Uses احنا and إنت. Shorter sentences, stronger verbs.

## Forbidden Words (Never Use)

### MSA Formality (Too Formal for Brand Voice)

| Word | Why | Instead Use |
|------|-----|-------------|
| يرجى | Government/official tone | "من فضلك" or just the imperative politely |
| نأسف | Robotic, cold | "معلش" — warmer, more human |
| بناءً على | Legalistic | "زي ما طلبت" / "ع حسب" |
| وبالتالي | Essay-like | عشان كده / يبقى |
| تجدر الإشارة | News report | "كمان مهم إن..." |
| غير أن / إلا أن | Written, not spoken | "بس" / "لكن" |
| حيث أن | Formal connector | عشان / لأن |
| مما يؤدي إلى | Overly complex | "وده بيخلي..." |
| سوف | MSA future | حـ prefix |
| سيتم | MSA passive | هنـ + verb (active voice) |

### Street / Slang (Too Casual for Brand Voice)

| Word | Why | Instead Use |
|------|-----|-------------|
| فشخ | Vulgar | أوي / قوي |
| بضان | Vulgar | Avoid entirely |
| جامد (as slang) | Can be offensive | قوي / ممتاز / حلو أوي |
| يا عم | Too familiar | حضرتك / أستاذ |
| بتاع (excessive use) | Overused filler | Use possessive suffixes instead |
| إيه ده كله | Sarcastic, dismissive | Avoid in client-facing content |
