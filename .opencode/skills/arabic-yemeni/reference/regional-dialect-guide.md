# Yemeni Arabic — Regional Dialect Guide

## Major Dialect Zones

| Zone | Area | Key Features |
|------|------|-------------|
| Sana'ani (Highlands) | Sana'a, Dhamar, Hajjah | Qaf → g, Jīm → ʒ, Conservative |
| Adenese (Southern Coastal) | Aden, Lahij | Qaf → q or hamza, Indian Ocean vocabulary |
| Hadrami | Hadramawt (East) | Qaf → q, Jīm → y, Unique vocabulary |
| Tihami (Coastal plain) | Hudaydah, Zabid | Qaf → g or k, African vocabulary |
| Ta'izzi | Ta'iz | Transitional between North and South |

## Sana'ani (Highland Yemeni)

| Feature | Example |
|---------|---------|
| Qaf | g (strong) — قلم → گلم |
| Jīm | j (preserved classical) |
| Kaf-in-context | Not palatalized |
| "I want" | أبغى (abghā) / أشتي (ashtī) |
| "What" | ماذا (mādhā) / إيش (aysh) |
| "Yes" | أي (ay) / إيوه (ēwa) |
| Intonation | Rhythmic, rising at sentence end |

## Adenese (Southern Yemeni)

| Feature | Example |
|---------|---------|
| Qaf | q (standard) or hamza |
| Jīm | ʒ (zh — like Cairo) |
| "I want" | أبي (abī) / أريد (arīd) |
| "What" | شنو (shinu) — shared with Gulf |
| "How" | كيف (kayf) |
| Vocabulary | English: باص (bus — bus), كمبيوتر (computer); Hindi: بغالة (bughāla — vessel) |

## Hadrami

| Feature | Example |
|---------|---------|
| Qaf | q (pre-Islamic preservation) |
| Jīm → y | جمل → يمل (yamil — camel) |
| Verb suffix -k | In perfect tense: رحت → رحتك (raḥt-k) |
| "I want" | أشتي (ashtī) |
| "Where" | أين (ayn) / فين (fayn) |
| Vocabulary | Somali/Omani influence from historical trade |
| Emphatic consonants | Very pronounced |
