# Balaghah — Arabic Rhetorical Devices

## Ma'ani (Syntax-Based Rhetoric)

| Device | Arabic | Effect | Example |
|--------|--------|--------|---------|
| Declarative | جملة خبرية | Statement of fact | "الله غفور رحيم" |
| Performative | جملة إنشائية | Command/question/wish | "أقيموا الصلاة" |
| Emphasis | توكيد | Strengthening claim | "والله لأفعلنَّ" |
| Restriction | قصر | Limiting attribute | "إنما المؤمنون إخوة" |
| Brevity | إيجاز | Condensed expression | "العلم نور" |

## Bayan (Figurative Language)

| Device | Arabic | Description |
|--------|--------|-------------|
| Simile | تشبيه | Explicit comparison using ك/مثل/كأن |
| Metaphor | استعارة | Implicit comparison (تصريحية / مكنية) |
| Metonymy | كناية | Indirect reference to attribute |
| Synecdoche | مجاز مرسل | Part for whole, cause for effect |

## Badi' (Embellishment)

| Device | Arabic | Example |
|--------|--------|---------|
| Paronomasia | جناس | "والسماء ذات الرجع" (r, j sound play) |
| Antithesis | طباق | "يُخرج الحي من الميت" |
| Contrast | مقابلة | "فأما من أعطى واتقى…" |
| Rhymed prose | سجع | Used in Quran and classical prose |
