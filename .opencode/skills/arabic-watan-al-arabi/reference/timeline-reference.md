# Al-Watan al-Arabi — Chronological Timeline Reference

## Pre-Islamic Era (before 610 CE)

| Period | Event |
|--------|-------|
| ~2000 BCE | Semitic migration from Arabian peninsula into Mesopotamia / Levant |
| ~1000–700 BCE | Kingdom of Saba (Sheba) in Yemen; Marib Dam; frankincense trade |
| 750–300 BCE | Kingdoms of Lihyan, Dedan, Kinda in north/northwest Arabia |
| 300 BCE–300 CE | Nabatean Kingdom (Petra); Palmyra (Tadmur) in Syria |
| 300–500 CE | Ghassanids (Roman vassals in Levant); Lakhmids (Persian vassals in Iraq) |
| 500–610 CE | Quraysh dominate Mecca; Pre-Islamic poetry (Mu'allaqat); Jahiliyyah period |
| ~570 CE | Year of the Elephant — Abraha attacks Mecca |
| 610 CE | First Quranic revelation to Prophet Muhammad |

## Islamic Golden Age & Classical Empires

| Period | Major Developments |
|--------|-------------------|
| 632–661 | Rashidun Caliphate — expansion to Levant, Egypt, Mesopotamia, Persia |
| 661–750 | Umayyad Caliphate (Damascus) — expansion to Spain, Central Asia; Arabic as administrative language |
| 750–1258 | Abbasid Caliphate (Baghdad) — Bayt al-Hikma; translation movement; golden age of science |
| 909–1171 | Fatimid Caliphate (Cairo) — Al-Azhar founded; Ismaili dynasty |
| 1095–1291 | Crusades; Saladin & Ayyubid recapture of Jerusalem (1187) |
| 1250–1517 | Mamluk Sultanate (Cairo) — defeated Mongols; trade prosperity |
| 1299–1922 | Ottoman Empire — most of Arab world under Ottoman control from 1517 |

## Colonial Period (1798–1946)

| Date | Event |
|------|-------|
| 1798 | Napoleon invades Egypt — start of modern colonialism |
| 1805 | Muhammad Ali Pasha rules Egypt — modernization |
| 1830 | France invades Algeria |
| 1881 | France establishes protectorate over Tunisia |
| 1882 | British occupation of Egypt |
| 1912 | Italy colonizes Libya |
| 1916 | Sykes-Picot Agreement — secret UK/France partition of Ottoman Arab provinces |
| 1917 | Balfour Declaration |
| 1918 | Collapse of Ottoman Empire in WWI |
| 1920 | French mandate over Syria & Lebanon; British mandate over Iraq, Palestine, Jordan |
| 1932 | Kingdom of Saudi Arabia founded; Iraq independent |
| 1943 | Lebanon independent |
| 1946 | Syria, Jordan independent |

## Post-Independence & Modern Era (1946–present)

| Date | Event |
|------|-------|
| 1945 | Arab League founded (Cairo) — 7 members |
| 1948 | Nakba — Palestinian exodus; Israel established |
| 1952 | Egyptian Revolution (Free Officers) |
| 1956 | Suez Crisis; Egypt nationalizes canal |
| 1958–1961 | United Arab Republic (Egypt + Syria) — collapse |
| 1967 | Naksa — Six-Day War; Israel occupies Sinai, Golan, West Bank |
| 1970s | Oil boom; Gulf state independence (1971: UAE, Qatar, Bahrain) |
| 1973 | October War (Yom Kippur) |
| 1979 | Egypt–Israel Peace Treaty; Iranian Revolution (non-Arab but affects region) |
| 1990–91 | Gulf War — Iraq invades Kuwait; US-led coalition |
| 2003 | US invasion of Iraq |
| 2005 | Cedar Revolution in Lebanon |
| 2010–2012 | Arab Spring — Tunisia, Egypt, Libya, Yemen, Syria uprisings |
| 2014 | Daesh/ISIS establishes caliphate in Iraq/Syria |
| 2017–2021 | Abraham Accords (UAE, Bahrain, Morocco, Sudan normalize with Israel); Houthi control of Yemen |
| 2023 | Gaza War (Oct 7 + aftermath); normalization disruptions |
