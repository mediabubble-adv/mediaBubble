# Arabic Dialectal Variations — Diglossia Reference

## The Diglossia Spectrum

| Level | Name | Used In | Features |
|-------|------|---------|----------|
| L1 | Pure dialect | Home, street, intimate | Full phonological shift; unique vocabulary; no MSA |
| L2 | Dialect with MSA borrowings | Social media, informal chats | MSA technical terms in dialect frame |
| L3 | Educated Spoken Arabic (ESA) | TV interviews, formal conversation | Dialect grammar + MSA vocabulary; most flexible |
| L4 | MSA with dialect coloring | News, official speeches | Full MSA grammar with slight phonological/dialect influence |
| L5 | Pure MSA (Fusha) | Written media, legal, Quran | Full case endings; classical grammar |

## Register Shift Triggers

| Context | Shift Direction | Example |
|---------|----------------|---------|
| Talking to older relative | L1 → L2/L3 | Add polite particles, remove slang |
| Talking about technology | L1 → L2 (borrow MSA/English terms) | "الجهاز دا عنده مشكلة في الـ connection" |
| Quoting religion | L1 → L5 | Switch to Quranic/hadith Arabic |
| Making a formal request | L1 → L3/L4 | Use MSA request patterns |
| Expressing emotion | L3/L4 → L1 | Anger/surprise triggers native dialect |
| Telling a joke | L3/L4 → L1 | Dialect is funnier and more authentic for humor |

## Content Creation — Dialect Strategy Quick Reference

| Target Audience | Default Register | Acceptable Mix |
|-----------------|-----------------|----------------|
| Egypt — mass market | Masri L2–L3 | MSA for religion, tech; English for brands |
| Levant — urban youth | Levantine L2 | English loanwords; little MSA |
| Gulf — formal | Khaliji L3 | MSA for government, finance; English for tech |
| North Africa | Darija L2–L3 | French words; MSA for religion |
| Pan-Arab | L3–L4 (ESA) | Dialect flavor without alienating other regions |
